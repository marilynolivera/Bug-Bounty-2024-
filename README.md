# Bug-Bounty-2024-
Proyecto de Bug Bounty. Análisis de vulnerabilidades e investigación profunda acerca de una empresa con programa de Bug Bounty. Desarrollo de informe con apartado técnico(procedimiento detallado de hallazgos) y gerencial (Resumen de los hallazgos importante en lenguaje no técnico, información para tomar acciones).
# Herramientas de Enumeración y Escaneo
  Google Dorks (Búsquedas avanzadas): Utilizadas para explorar dominios, subdominios, correos electrónicos y documentos públicos.
  Sublist3r: Herramienta empleada específicamente para enumerar subdominios del dominio objetivo.
  Nmap: Utilizada para escanear la red, identificar dispositivos, servicios, puertos abiertos y sistemas operativos.
  TheHarvester: Empleada para recolectar correos electrónicos, subdominios, direcciones IP y nombres de empleados de fuentes públicas.
  Recon-ng: Marco de reconocimiento utilizado para ejecutar diversos módulos de búsqueda de contactos, empresas y perfiles de usuario.
  Curl: Usada para solicitar cabeceras HTTP y obtener información adicional sobre los servidores web.
  Buscadores Técnicos y Bases de Datos
  Shodan: Utilizado para encontrar dispositivos y servicios directamente asociados al dominio.
  Censys: Herramienta de búsqueda para descubrir dispositivos y certificados SSL/TLS asociados.
  Whois: Consultada para identificar a los gestores del dominio y de los bloques de direcciones IP.
  Dig: Utilizada para realizar consultas DNS detalladas, incluyendo registros MX, NS, SRV y transferencias de zona.
  BGP (bgp.he.net): Consultada para obtener información sobre el Sistema Autónomo (AS), registros DNS y bloques IP.
  RDAP (client.rdap.org): Utilizada para confirmar información de registro de dominios e IPs.
# Análisis de Filtraciones y Seguridad
  DeHashed: Herramienta de análisis de brechas de seguridad y filtraciones de datos.
  Have I Been Pwned: Consultada para verificar si los correos electrónicos encontrados figuran en bases de datos filtradas.
  Breach Directory: Utilizada para buscar información sobre filtraciones de seguridad.
  VerifyEmailAddress.org: Empleada para verificar la validez de las direcciones de correo recolectadas.
  VirusTotal: Utilizada para analizar enlaces sospechosos antes de acceder a ellos.
# Otras Fuentes de Información
  LinkedIn: Consultada para obtener información sobre la empresa, empleados y jerarquías organizacionales.
  GitHub: Investigada para encontrar repositorios públicos, posibles credenciales expuestas y nombres de colaboradores.
  Surfshark (VPN): Utilizada como medida de precaución durante los escaneos intensivos para evitar bloqueos de IP.
